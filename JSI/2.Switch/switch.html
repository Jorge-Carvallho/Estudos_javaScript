Estrutura switch em JavaScript

A estrutura switch em JavaScript é utilizada para executar diferentes blocos de código com base no valor de uma expressão. É uma alternativa ao uso de múltiplos if e else if, especialmente quando se lida com várias condições baseadas em valores discretos.

Conceito Básico:

  • switch: Avalia uma expressão e compara seu valor com diferentes casos (case).
  • case: Define um valor específico para comparar com a expressão.
  • default: Opcionalmente, especifica o bloco de código a ser executado se nenhum dos casos corresponder.

Funcionamento:

  1. Expressão: O switch avalia uma expressão e compara o resultado com os valores especificados nos case.
  2. Casos: Se o valor da expressão corresponder a um case, o bloco de código correspondente será executado.
  3. Padrão: Se nenhum case corresponder, o bloco de código no default será executado (se estiver presente).

Exemplo Prático:


const dia = 'quarta-feira';

switch (dia) {
    case 'segunda-feira':
        console.log('Início da semana de trabalho.');
        break;
    case 'quarta-feira':
        console.log('Dia de reunião.');
        break;
    case 'sexta-feira':
        console.log('Preparação para o final de semana.');
        break;
    default:
        console.log('Dia não reconhecido.');
}

Aplicações Comuns:

  • Escolhas Baseadas em Valor: Quando você precisa executar diferentes ações dependendo de um valor específico.
  • Organização de Código: Facilita a leitura e manutenção do código em comparação com múltiplos if e else if.
