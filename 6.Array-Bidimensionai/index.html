Exemplo de Arrays Bidimensionais em JavaScript

Abaixo está um exemplo que percorre uma matriz bidimensional, exibe os valores e calcula a soma de todos os elementos:


// Definindo uma matriz bidimensional de números
const matrizNumeros = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9],
];

// Exibindo a matriz original na página
document.getElementById('matrizOriginal').textContent = JSON.stringify(matrizNumeros);

// Variável para armazenar a soma dos elementos
let soma = 0;
let resultado = '';

// Iterando sobre a matriz bidimensional
for (let i = 0; i < matrizNumeros.length; i++) {
    for (let j = 0; j < matrizNumeros[i].length; j++) {
        const valor = matrizNumeros[i][j];
        soma += valor; // Somando o valor atual à variável soma
        resultado += 'Posição (' + i + ',' + j + ') tem o valor: ' + valor + '
';
    }
}

// Exibindo o resultado na página
document.getElementById('resultado').innerHTML = resultado;
document.getElementById('somaTotal').textContent = soma;
