Exemplo de Manipulação de Arrays em JavaScript

Introdução

Este exemplo demonstra várias operações comuns em arrays, como adicionar, remover, concatenar e iterar sobre elementos em JavaScript.

Operações Demonstradas

  • Adicionar elementos ao final do array com push
  • Adicionar elementos ao início do array com unshift
  • Remover o último elemento com pop
  • Remover o primeiro elemento com shift
  • Pesquisar elementos com includes e indexOf
  • Criar cópias parciais com slice
  • Concatenar arrays com concat
  • Remover e substituir elementos com splice
  • Iterar sobre os elementos do array com for loop

Conclusão

Essas operações são essenciais para a manipulação de arrays em JavaScript e permitem flexibilidade ao trabalhar com listas de dados.
