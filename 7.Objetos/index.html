Exemplo de Manipulação de Objetos em JavaScript

Introdução

Este exemplo demonstra como criar e manipular objetos em JavaScript, atribuindo propriedades e criando objetos aninhados.

Operações Demonstradas

  • Criar um objeto vazio
  • Atribuir propriedades ao objeto
  • Adicionar propriedades aninhadas (objetos dentro de objetos)
  • Exibir o objeto no console

Detalhamento das Operações

  • Criação de Objeto: O objeto pessoa foi criado vazio e posteriormente foram adicionadas propriedades como nome, idade e trabalho.
  • Adição de Endereço: Uma nova propriedade chamada endereco foi atribuída ao objeto pessoa, contendo um outro objeto com as propriedades rua, número, e bairro.

Conclusão

Manipular objetos em JavaScript permite organizar dados em estruturas complexas, facilitando o acesso e a modificação das informações. É uma das ferramentas mais poderosas no desenvolvimento web.
